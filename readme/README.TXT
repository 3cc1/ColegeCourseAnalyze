College Course Analyze
  An AI-Powered college transcript analyzing tool. 
  Helps with advising student's academic plans from PDF files and generating personolized graduation plan using Gemini API.
What it does:
  Extracting student data from PDF transcript
  Analyzing Student Gradings and Progress
  Helps with future class schedule
  Graduation timeline estimate
  Academic Advice

Setup Instructions:
1. Clone the repository:
"git clone https://github.com/3cc1/ColegeCourseAnalyze.git
cd ColegeCourseAnalyze:
2. Create a virtual environment:
python -m venv .venv
.venv\Scripts\activate   # Windows
Install Dependencies:
pip install -r requirements.txt
4. Add environment variables:
GEMINI_API_KEY=your_api_key_here
